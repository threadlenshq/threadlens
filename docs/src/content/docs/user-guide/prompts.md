Prompts define the response templates that marketing mode uses for engagement workflows. They are only available after graduating a research project to marketing mode.

## When prompts are used

Prompts are used when ThreadLens acts on leads in marketing mode: drafting replies, generating karma-friendly comments, sending DMs, and crafting product-related outreach on Reddit and Bluesky.

## Prompt types

| Type | Platform | Purpose |
| --- | --- | --- |
| Product | Reddit | Product announcement and outreach replies |
| Karma | Reddit | Low-risk karma-building comments |
| DM | Reddit | Direct message templates |
| Product | Bluesky | Product announcement and outreach replies |
| Karma | Bluesky | Low-risk karma-building comments |

## Editing prompts

Click a prompt header in the Prompts editor to expand and edit the template text. Changes save automatically on blur. If a prompt type has no template yet, click **Add prompt** to create one.

Set prompts after the project graduate config has been generated so the product angle and market context are available.

For graduation and modes, see [Projects and Modes](./projects-and-modes/).
